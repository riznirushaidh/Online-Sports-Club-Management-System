# Online Sports Club Management System

A web-based application built with PHP and MySQL to manage sports club activities.

## Features
- Admin Dashboard (Manage Events, Equipment, Members)
- Member Registration & Login
- Event Booking System
- Equipment Borrowing with Inventory Tracking
- Messaging System

## How to Run
1. Install XAMPP.
2. Download this project and put the folder in `C:\xampp\htdocs`.
3. Open phpMyAdmin and create a database named `sports_club_db`.
4. Import the `sports_club_db.sql` file found in this folder.
5. Open your browser to `http://localhost/sports_club`.
